### Solution to the backend interns challenge


Running the functions in main

- Open the terminal
- Navigate to the project
  
      Cd src
      Cd holidaycalendar

      dotnet run

- Open the terminal
- Navigate to the project
  
      Cd tests
      Cd holidaycalendar.tests
- Run the test class
  
      dotnet test
